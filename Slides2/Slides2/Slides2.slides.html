What is Machine Learning?¶

A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P, if its performance at tasks in T, as measured by P, improves with experience E.

  • Mitchell, Tom. M. 1997. Machine Learning. New York: McGraw-Hill

What is Machine Learning?¶

Things learn when they change their behavior in a way that makes them perform better in the future.

  • Witten, Ian H., and Eibe Frank. 2000. Data Mining: Practical Machine Learning Tools and Techniques with Java Implementations. San Diego, CA: Morgan Kaufmann.

What is Machine Learning about?¶

  • DATA (sometimes a lot of them):
    • Gathering them
    • Analyzing them
  • Algorithms:
    • Choose the algorithms
    • Program them
  • Metric
    • Choose the appropriate one
    • Use it to evaluate the performance

Different types of learning in this lecture¶

  • Supervised Learning: Learns a mapping from some input features to a known output
    • Prediction of a feature from the others, e.g. class of the flowers (classification)
    • Learn a function that explains your data (regression)
  • Unsupervised Learning: Learns to represent a set of input without any feedback
    • Find a subset of features that represent most of your data (PCA)
    • Find group of samples based on their features (clustering)

Take Home Message¶

  • Always look at your data before anything else
    • Examine their dimensions
    • Plot the repartition of features, classes (if provided)
  • Define what you want to learn from the data
    • Predicting a certain feature (supervised learning)
    • Clustering your samples (unsupervised learning)
  • Choose a metric to evaluate the performances of your algorithms
