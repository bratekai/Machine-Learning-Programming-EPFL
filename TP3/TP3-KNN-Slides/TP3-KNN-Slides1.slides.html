KNN Algorithm¶

  • A supervised learning algorithm (finally!)
  • Used for classification and regression
  • Based on a metric between samples

Some definitions¶

  • KNN is non-parametric
    • No assumption on the underlying data distribution
    • Perfect to learn arbitrary datasets
  • KNN is lazy
    • No generalization from the training data
    • All training samples are stored and used for query
